# Инструкция по настройке AI сервисов

## 1. Получение API ключей

### OpenRouter API Key
1. Перейдите на https://openrouter.ai/
2. Зарегистрируйтесь или войдите в аккаунт
3. Перейдите в раздел "Keys" в настройках
4. Создайте новый API ключ
5. Скопируйте ключ

### OpenAI API Key
1. Перейдите на https://platform.openai.com/
2. Войдите в аккаунт или создайте новый
3. Перейдите в раздел "API keys"
4. Создайте новый секретный ключ
5. Скопируйте ключ

## 2. Настройка проекта

1. Откройте файл `.env.local`
2. Замените placeholder'ы на ваши реальные API ключи:

```env
OPENROUTER_API_KEY=your_actual_openrouter_api_key_here
OPENAI_API_KEY=your_actual_openai_api_key_here
```

3. Сохраните файл

## 3. Запуск проекта

```bash
# Установка зависимостей (если еще не установлены)
npm install

# Запуск в режиме разработки
npm run dev
```

Откройте http://localhost:3000 в браузере.

## 4. Как это работает

### Анализ стиля референсов
1. Загрузите референсы (изображения/видео) в проект
2. Нажмите "Анализировать стиль"
3. AI **реально анализирует изображения** с помощью Vision API (GPT-4o) и определяет:
   - **Доминирующие цвета** - извлекаются напрямую из изображений
   - **Настроение контента** - анализируется визуальный стиль
   - **Темп (slow/medium/fast)** - определяется по динамике сцены
   - **Визуальный стиль** - minimalist, vintage, modern, и т.д.
   - **Объекты и сцены** - распознаются элементы на изображениях

**Важно:** AI анализирует только изображения (image файлы). Видео и аудио файлы учитываются по названиям.

### Генерация контента
1. Перейдите на страницу генерации
2. Опишите тему контента
3. AI создаст:
   - Заголовок
   - Описание
   - Сценарий
   - Теги
   - Промпт для генерации видео

### Публикация
1. Подключите социальные сети в разделе "Площадки"
2. Перейдите на страницу публикации
3. Выберите платформы
4. Нажмите "Опубликовать"

## 6. AI Модели

Проект использует:
- **OpenRouter**: meta-llama/llama-3.1-70b-instruct для анализа и генерации
- **OpenAI**: gpt-4o для анализа и генерации, dall-e-3 для изображений
- **Stability AI**: для генерации изображений (уже настроено)

Система автоматически выбирает провайдер:
- Если есть OPENAI_API_KEY - использует OpenAI
- Иначе использует OpenRouter

## 7. Troubleshooting

### Ошибка "API key not configured"
- Убедитесь, что API ключи правильно указаны в `.env.local`
- Перезапустите сервер разработки

### Ошибка генерации контента
- Проверьте баланс API ключа
- Убедитесь, что интернет-соединение стабильно
- Проверьте логи в консоли

### Ошибка публикации
- Проверьте подключение платформ
- Убедитесь, что credentials платформ правильные
- Для n8n webhook проверьте URL webhook
